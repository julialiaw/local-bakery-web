# local-bakery-web
just a cute page for your local bakery with custom hand-drawn fonts.

## how to run it
you can check out [this link](https://localbakeryweb.netlify.app/) or download the files and run it yourself.

## why this webpage?
Since I'm new to coding, obviously a webpage is an easy starter. I really wanted to learn things like having a fixed header/navigation bar, making multiple pages, and making the website responsive so it's still cool on mobile. There wasn't much thought behind making it a website for a bakery; i just thought it would be fun and it was easy to fine a color palette that fit the website. As for the fonts, honestly it would've been a lot more readable with a standard sans-serif font but it was just a small thing i felt like throwing in.
